## ğŸš€ Fitur Website
Website ini terdiri dari dua halaman utama dengan fitur sebagai berikut:

### 1. Halaman Utama (index.html)
Halaman satu layar (*Single Page*) dengan navigasi anchor yang mencakup:
* **Hero Section:** Tagline "Smart Farming for a Better Future" dengan tombol Call-to-Action.
* **Quick Stats:** Statistik kunjungan, lokasi kebun, dan program edukasi.
* **About Us:** Sejarah singkat, Visi & Misi, serta profil tim.
* **Programs & Services:** Tabel daftar harga dan durasi program edukasi.
* **Facilities:** Daftar fasilitas pendukung dan galeri foto kebun.
* **Location & Contact:** Informasi detail 2 lokasi kebun (Tasikmalaya & Garut).

### 2. Halaman Pendaftaran (registration.html)
Halaman khusus pendaftaran kunjungan yang mencakup:
* **Formulir Lengkap:** Validasi input untuk nama, email, telepon, instansi, tanggal, jumlah peserta, program, dan lokasi.
* **Sidebar Info:** Kontak dukungan admin dan FAQ sederhana untuk calon pengunjung.
